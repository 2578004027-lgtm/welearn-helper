# WeLearn 网课助手 v1.0

WeLearn 在线学习辅助工具。

## 快速开始

```bash
pip install requests colorama cryptography
python main.py
```

启动后选择模式：

| 模式 | 说明 |
|------|------|
| 课程模式 | 自动完成练习题，自定义正确率（固定/随机） |
| 时长模式 | 多线程模拟学习时间 |
